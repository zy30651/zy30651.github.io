可迭代对象 ​

返回一个迭代器的对象。本质是具备了__iter__方法的对象；

常见的可迭代对象有：list、set、dict、tuple、str

py
from collections import Iterable
isinstance([], Iterable) # True

自定义可迭代对象

py
class IterClass():
	def __iter__(self):
		pass
c1=IterClass()
isinstance(c1, Iterable)

迭代器(Iterator) ​

调用next()方法时可以返回可迭代对象的下一个值的对象，为迭代器。

本质是实现了__iter__和__next__方法的对象

iter：返回迭代器本身

next：返回下一个值，如果下一个值不存在，则报错

py
class MyNumberIterator:
    def __init__(self, start, end):
        self.current = start
        self.end = end

    def __iter__(self):
        return self

    def __next__(self):
        if self.current < self.end:
            num = self.current
            self.current += 1
            return num
        else:
            raise StopIteration

# 使用自定义迭代器
my_iterator = MyNumberIterator(1, 5)
for num in my_iterator:
    print(num)

生成器(generator) ​

一次生成一个值的特殊类型函数；使用yield的函数；是特殊的迭代器

生成器就是一个迭代器在调用过生成器运行的过程中，每次遇到yield时，函数会暂停并保存当前所有的运行信息，返回yield的值，并在下一次执行next()方法时，从当前位置继续执行

py
b = (i for i in range(3)) # <generator object <genexpr> at 0x101ac85f0>
next(b) # 0
next(b) # 1
C = (i for i in range(3))
for x in c:
  print(x) 
# 0 1 2

使用生成器写斐波那契数列

py
# 方法1
def fibo(num):
	numlist=[0,1]
	for i in range(num-2):
		numlist.append(numlist[-2]+numlist[-1])
  return numlist
# 方法2 yield
def fab(max):
  n,a,b=0,0,1
  while n<max:
    yield b
    a,b=b,a+b
    n=n+1
for n in fab(5):
  print(n)

next()和send()的区别 ​

next()用于生成器中获取下一个值；会自动启动生成器

py
def my_generator():
    yield 1
    yield 2
    yield 3

gen = my_generator()
print(next(gen))  # 输出: 1
print(next(gen))  # 输出: 2
print(next(gen))  # 输出: 3
try:
    print(next(gen))
except StopIteration:
    print("生成器已结束")

send()方法不仅可以让生成器从上次暂停位置继续执行，还可以传值；第一次调用send()参数必须为None

py
def generator():
	print("生成器启动")
	value = yield "初始化结果"
	while True:
		print(f"接受到的值为{value}")
		value = yield f"处理后的值为{value*2}"
gen=generator()

print(next(gen))# 生成器启动  初始化结果
print(gen.send(10)) #　接受到的值为10　处理后的值为20

print(gen1.send(None))　# 生成器启动 初始化结果
