装饰器 ​

一、简述 ​

装饰器能在不修改原有函数代码的基础上，对函数的功能进行扩展。这不仅提高了代码的复用性，还让代码结构更加清晰、易于维护。

二、装饰器的基本概念 ​

（一）函数作为对象 ​

在 Python 中，函数可以作为变量被赋值、作为参数传递给其他函数，也能作为返回值返回。这是装饰器实现的基础。

python
def greet():
    return "Hello!"

# 将函数赋值给变量
say_hello = greet
print(say_hello())
（二）闭包 ​

闭包是指有权访问另一个函数作用域中变量的函数。在装饰器的实现中，闭包起到了关键作用。

python
def outer_function(message):
    def inner_function():
        return message
    return inner_function

# 创建闭包
greet_message = outer_function("Hi!")
print(greet_message())

三、装饰器的基本语法 ​

（一）简单装饰器 ​

下面是一个简单的装饰器示例，用于在函数执行前后打印日志：

python
def log_decorator(func):
    def wrapper():
        print(f"开始执行 {func.__name__} 函数")
        result = func()
        print(f"{func.__name__} 函数执行结束")
        return result
    return wrapper

@log_decorator
def say_hi():
    print("Hi!")

say_hi()

log_decorator 是一个装饰器函数，它接受一个函数作为参数，并返回一个新的函数 wrapper。@log_decorator 语法糖等价于 say_hi = log_decorator(say_hi)。

（二）带参数的装饰器 ​

如果被装饰的函数需要接受参数，那么装饰器中的 wrapper 函数也需要接受相应的参数。

python
def log_decorator(func):
    def wrapper(*args, **kwargs):
        print(f"开始执行 {func.__name__} 函数")
        result = func(*args, **kwargs)
        print(f"{func.__name__} 函数执行结束")
        return result
    return wrapper

@log_decorator
def add(a, b):
    return a + b

print(add(3, 5))
（三）带参数的装饰器工厂 ​
python
def repeat(num):
    def decorator(func):
        def wrapper(*args, **kwargs):
            result = None
            for _ in range(num):
                result = func(*args, **kwargs)
            return result
        return wrapper
    return decorator

@repeat(3)
def greet():
    print("Hello!")

greet()

在这个例子中，repeat 是一个装饰器工厂，它接受一个参数 num，并返回一个装饰器 decorator。

四、装饰器的应用场景 ​

（一）日志记录 ​

除了前面提到的简单日志记录，装饰器还可以用于详细的日志记录，如记录函数的执行时间、输入输出等。

python
import time

def log_time(func):
    def wrapper(*args, **kwargs):
        start_time = time.time()
        result = func(*args, **kwargs)
        end_time = time.time()
        print(f"{func.__name__} 函数执行时间: {end_time - start_time} 秒")
        return result
    return wrapper

@log_time
def long_running_task():
    time.sleep(2)

long_running_task()
（二）权限验证 ​

在 Web 开发中，装饰器可以用于验证用户的权限，只有具备相应权限的用户才能访问某些接口。

python
def requires_permission(permission):
    def decorator(func):
        def wrapper(*args, **kwargs):
            # 模拟权限验证
            user_permissions = ['admin']
            if permission in user_permissions:
                return func(*args, **kwargs)
            else:
                print("权限不足")
        return wrapper
    return decorator

@requires_permission('admin')
def admin_function():
    print("执行管理员操作")

admin_function()
（三）缓存 ​

装饰器可以实现简单的缓存功能，避免重复计算。

python
def cache_decorator(func):
    cache = {}
    def wrapper(*args):
        if args in cache:
            return cache[args]
        result = func(*args)
        cache[args] = result
        return result
    return wrapper

@cache_decorator
def factorial(n):
    if n == 0 or n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))
print(factorial(5))
（四）单例 ​
py
def singleton(cls):
    # 用于存储每个被装饰类的唯一实例
    instances = {}
    def wrapper(*args, **kwargs):
        # 检查该类是否已经有实例存在
        if cls not in instances:
            # 如果没有实例，则创建一个新实例并存储在字典中
            instances[cls] = cls(*args, **kwargs)
        # 返回该类的唯一实例
        return instances[cls]
    return wrapper

@singleton
class MyClass:
    def __init__(self):
        print("MyClass 实例已创建")

# 创建第一个实例
obj1 = MyClass()
# 创建第二个实例
obj2 = MyClass()

# 检查两个实例是否为同一个对象
print(obj1 is obj2)

五、装饰器的注意事项 ​

（一）函数元信息丢失 ​

使用装饰器后，被装饰函数的元信息（如 __name__、__doc__ 等）会被 wrapper 函数的元信息替代。可以使用 functools.wraps 来解决这个问题。

python
import functools

def log_decorator(func):
    @functools.wraps(func)
    def wrapper():
        print(f"开始执行 {func.__name__} 函数")
        result = func()
        print(f"{func.__name__} 函数执行结束")
        return result
    return wrapper

@log_decorator
def say_hi():
    """这是一个打招呼的函数"""
    print("Hi!")

print(say_hi.__name__)  
print(say_hi.__doc__)
（二）多个装饰器的执行顺序 ​

当一个函数被多个装饰器装饰时，装饰器的执行顺序是从下往上的。

python
def decorator1(func):
    def wrapper():
        print("装饰器 1 开始")
        result = func()
        print("装饰器 1 结束")
        return result
    return wrapper

def decorator2(func):
    def wrapper():
        print("装饰器 2 开始")
        result = func()
        print("装饰器 2 结束")
        return result
    return wrapper

@decorator1
@decorator2
def test_function():
    print("执行测试函数")

test_function()
