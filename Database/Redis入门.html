Redis 入门 ​

简介 ​

Redis（Remote Dictionary Server）是一个开源的、基于内存的数据结构存储系统，它可以用作数据库、缓存和消息中间件

支持多种数据结构，如 String、Hash、List、Set、Sorted Set 等

安装 ​

  1. MacOS

    bash
    brew install redis

  2. Ubuntu

    bash
    sudo apt update
sudo apt install redis-server

启动 ​

  1. MacOS

    bash
    redis-server # 服务器

  2. Linux

    bash
    systemctl start redis-server  # 启动服务器
systemctl enable redis-server # 开机自启动
systemctl status redis-server # 检查服务状态

连接 ​

bash
redis-cli # 客户端
redis-cli -h hostname -p port

常用命令 ​

5.1 键（Key）操作 ​
  • 查看所有键
bash
KEYS *

注意：在生产环境中，由于 KEYS 命令可能会阻塞 Redis 服务器，建议谨慎使用。可以使用 SCAN 命令进行渐进式遍历。

  • 检查键是否存在
bash
EXISTS key1

若键存在返回 1，否则返回 0。

  • 删除键
bash
DEL key1 key2 key3

可以同时删除多个键，如 DEL key1 key2 key3。

  • 设置键的过期时间（单位：秒）
bash
EXPIRE mykey 60 #  表示将键 `mykey` 的过期时间设置为 60 秒。
5.2 字符串（String）操作 ​
  • 设置字符串值
bash
SET name "John"
  • 获取字符串值**
bash
GET name
  • 递增数值
bash
INCR key1

如果键的值是一个整数，该命令会将其值加 1。如果键不存在，会先将其值初始化为 0 再进行递增。

  • 递减数值
bash
DECR key1

与 INCR 相反，将键的值减 1。

5.3 哈希（Hash）操作 ​
  • 设置哈希字段值
bash
HSET user:1 name "Alice" age 25

例如，HSET user:1 name "Alice" age 25。

  • 获取哈希字段值
bash
HGET user:1 name
  • 获取哈希所有字段和值
bash
HGETALL user:1
5.4 列表（List）操作 ​
  • 在列表头部插入元素
bash
LPUSH mylist "apple" "banana"
  • 在列表尾部插入元素
bash
RPUSH mylist "apple"
  • 获取列表指定范围的元素
bash
LRANGE mylist 0 -1 # 会返回列表 `mylist` 中的所有元素。
5.5 集合（Set）操作 ​
  • 向集合中添加元素
bash
SADD myset "red" "green" "blue"
  • 获取集合中的所有元素
bash
SMEMBERS myset
  • 判断元素是否在集合中
bash
SISMEMBER myset "red" # 若元素存在返回 1，否则返回 0。
5.6 有序集合（Sorted Set）操作 ​
  • 向有序集合中添加元素
bash
ZADD myzset 10 "apple" 20 "banana"
  • 获取有序集合指定排名范围的元素
bash
ZRANGE myzset 0 -1 # 会按分数从小到大返回有序集合 `myzset` 中的所有元素
